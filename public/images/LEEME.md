# Imágenes del portfolio

Pon aquí tus imágenes y actualiza las rutas en `src/data/media.ts`.

## Estructura esperada

```
public/
  images/
    hero-bg.jpg         → fondo de la página de inicio
    portrait.jpg        → tu foto en el hero
    about-photo.jpg     → tu foto en "Sobre mí"
    reel-poster.jpg     → miniatura del showreel
    projects/
      proyecto-1.jpg
      proyecto-2.jpg
      proyecto-3.jpg
      ...
  videos/
    hero.mp4            → vídeo de fondo (opcional)
    showreel.mp4        → showreel (opcional)
```

## Consejos

- Usa JPG para fotos y fondos (más ligero).
- Usa PNG si necesitas transparencia.
- Para el hero, una imagen apaisada de al menos 1920×1080px.
- Para proyectos, imágenes en formato 4:3 o 16:9 quedan mejor.
